# 2026-06-25 H4 小猪校徽 CRM Logo 更新

## 背景

课程项目原先使用本地绘制的深大风格 SVG 徽章作为 favicon、品牌标识和用户头像。为了让 Smart CRM 的视觉识别更贴合“智能销售副驾”主题，本次改为使用 H4 方向的小猪校徽 CRM 融合图标。

## 更新内容

- 使用 H4 方案生成透明 PNG：小猪主体、耳麦、`SZU`、CRM 趋势图和客户图标。
- 新增 `src/assets/smart-crm-logo.png`，作为 React 应用内品牌图和头像图。
- 新增 `public/favicon.png` 和 `public/smart-crm-logo.png`，作为浏览器 favicon 与 public 访问资源。
- 删除旧的 `public/favicon.svg` 和 `src/assets/szu-emblem.svg`。
- 调整品牌图、用户头像、个人资料头像的浅底和描边颜色，使其与新 logo 的蓝绿配色一致。

## 验证

- `npm run build` 通过。
- 已确认代码中不再引用旧 `szu-emblem.svg` 或 `favicon.svg`。
